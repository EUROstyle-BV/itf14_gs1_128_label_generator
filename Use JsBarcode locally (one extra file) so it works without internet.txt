How to do it?

19:00
The fix is straightforward — one extra file, one line change in index.html.

Step 1: Download JsBarcode

Go to this URL in your browser and save the file as JsBarcode.all.min.js in the same folder as index.html:

https://cdn.jsdelivr.net/npm/jsbarcode@3.11.6/dist/JsBarcode.all.min.js
(Right-click → Save as, or just let the browser download it.)

Step 2: Change one line in index.html

Find this line near the bottom:

html
<script src="https://cdn.jsdelivr.net/npm/jsbarcode@3.11.6/dist/JsBarcode.all.min.js"></script>
Replace it with:

html
<script src="JsBarcode.all.min.js"></script>
```

**That's it.** Your file structure becomes:
```
label-generator/
├── index.html
└── JsBarcode.all.min.js
Zip those two files together when selling.
